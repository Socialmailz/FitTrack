# FitTrack Mini

![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)

A simple and beautiful fitness tracking application built with Flutter. Track your daily steps, water intake, and other activities with ease. FitTrack Mini is designed to be fully functional offline, so you can track your progress anywhere, anytime.

## Features

*   **Dashboard:** A beautiful dashboard that provides a summary of your daily fitness activity, including a live step counter, water intake tracker, and recent activities.
*   **History:** View your activity history and track your progress over time.
*   **Create Activity:** Log your workouts and other fitness activities with a simple and intuitive interface.
*   **Analytics:** Visualize your fitness data with beautiful and interactive charts.
*   **Settings:** Customize your app experience with a flexible theme provider and other settings.
*   **Offline First:** All your data is stored locally on your device, so you can use the app without an internet connection.
*   **Light & Dark Mode:** Choose between a light or dark theme to suit your preference.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   Flutter SDK: [https://flutter.dev/docs/get-started/install](https://flutter.dev/docs/get-started/install)

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/your_username/fittrack_mini.git
    ```
2.  Install packages
    ```sh
    flutter pub get
    ```
3.  Run the app
    ```sh
    flutter run
    ```
